	- body 객체 : 대용량 문자열 등을 전송할 때 사용, 주소에 데이터를 기록하지 못하므로 새로고침이나 즐겨찾기 등을 사용할 수 없음

```
function <함수 이름>(<매개 변수>) {

}
```

```

```

```

function print(name, count) {  
	console.log(‘${name}이/가 ${count}개 있습니다.’)  
}  

// 함수를 시작합니다.  
print(“사과”, 10);  
print(“사과”);  
-> 실행하면 undefined가 출력됨  
```
