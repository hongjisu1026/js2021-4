__1. 함수 생성 방법__  
__※ 익명함수__ : 이름을 붙이지 않고 함수 생성
- 기본 형태
```			
 let <함수 이름> = function () { };
```
__※ 선언적함수__ : 이름을 붙여 함수를 생성 
- 기본 형태  
```
function <함수 이름> () { }
```
__※ 화살표함수__
- 하나의 표현식을 리턴하는 함수를 만들 때는 중괄호 생략 가능
- 기본 형태
```
() => {}
```
__2. 함수의 기본 형태__
```
function <함수 이름>(<매개 변수>) {  
	<함수 코드>  
	return <리턴 값>  
}
```


__3. 리턴하는 함수의 기본 형태__
- 기본 형태 
```  
function (<매개 변수>, <매개 변수>) {  
	let output = <초깃값>;  
	// output 계산  
	return output;  
} 
```

__4. 함수 매개 변수 초기화__  
- 매개 변수를 입력하지 않고 함수를 호출  
```
// 함수를 선언합니다.   
function print(name, count) {  
	console.log(‘${name}이/가 ${count}개 있습니다.’)  
}  

// 함수를 시작합니다.  
print(“사과”, 10);  
print(“사과”);  
-> 실행하면 undefined가 출력됨  
```
